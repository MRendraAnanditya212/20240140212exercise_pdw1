## Tema Project
Project ini adalah sebuah landing page bertema **Grand Theft Auto V (GTA 5) Online**. 
Website ini berfokus pada dunia bisnis di Los Santos, menampilkan interaksi karakter utama 
(seperti Franklin Clinton) dan fenomena unik (glitch) yang sering terjadi di area 
Bandara Internasional Los Santos (LSIA).

Website ini dibuat menggunakan HTML5, CSS3, dan sedikit sentuhan JavaScript untuk fitur interaktif.
